#필터 제거 후 accuracy 출력하는 코드 


데이터셋은 Imagenet -> Mnist -> Cifar10 사용했고, Imagenet -> Mnist 학습된 모델은 228 서버 rlaalsrl9876 폴더에서 model2.h5로 저장해놨어. 터미널에서 다음 코드 입력하면 될거야:


python3 -c 'from GA_pruning_paper import *; n2_model = load_(); accuracy2(n2_model, filter, conv_layer)' 


여기서 filter은 1,2,3,4번째 필터 유지하고 싶으면 [1,2,3,4]로 넣고, conv_layer은 0,1,4,5,8,9,10,13,14,15,18,19,20 이 중에서 넣으면 돼 (우린 VGG 사용해서 conv layer 총 13개)


예시: python3 -c 'from GA_pruning_paper import *; n2_model = load_(); accuracy2(n2_model, [1,2,3,4], 0)' 


그리고 첫번째층 동결시키고 모델 저장하고 싶으면:


python3 -c 'from GA_pruning_paper import *; n2_model = load_(); save_(accuracy2(n2_model, [1,2,3,4], 0))’


그 다음 층에선 그럼 load를 수정해서:


python3 -c 'from GA_pruning_paper import *; n2_model = load_checkpoint_(); accuracy2(n2_model, [5,6,7,8], 1))'



이렇게 하면 될 거야. ipynb 파일을 원하거나 다른 요청 사항 있으면 얘기해줘!
